# Jcink Category Port

A Discourse theme component redesigning the category view to a more familiar jcink format. Basically every feature present on jcink is now included in the category layout view.

Ended up having to make a custom fork of what is essentially the Discourse Category Groups component in order to use our own custom category layout.
