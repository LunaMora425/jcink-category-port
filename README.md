# Jcink Category Port

A Discourse theme component redesigning the category view to a more familiar jcink format. Basically every feature present on jcink is now included in the category layout view.

Ended up having to make a custom fork of what is essentially the Discourse Category Groups component in order to use our own custom category layout.

## Sources
Original Category Groups component: https://meta.discourse.org/t/category-groups/200874

## RP Resources Category

The RP Resources category, with its custom redirects and images, are included in this theme component. Disabling this theme component disables both the category layout and the RP Resources cards.
